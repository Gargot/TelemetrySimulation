executable	- два exe файла (клиент и сервер)
src		- исходники, файлы проекта для QtCreator

QtTelemetryServer - отображает последнии десять сообщений от клиентов, слушает порт 1234. 
Формат отображения:
	Value: полученное число/дата	Client address: адрес клиента	  Порт: 

QtTelemetryClient - отправлеяет сообщения серверу. Бокс "Server connection" отвечает за подключения к серверу,
отображает состояние текущего подключения и позвоялет подключится к другому серверу (при запуске подключает к localhost через 1234).
Бокс Data отображает полученное число от сервера, позволяет задать интервал отправки своих сообщений и 
отправлять дату. 

Допускается подключение множества клиентов к одному серверу. Сервер выделяет для каждого клиента отдельный поток.
